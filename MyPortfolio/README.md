# Virtual Assistant Portfolio

A modern, clean, and professional virtual assistant portfolio built with React, Vite, Tailwind CSS, Framer Motion, and React Icons. The site is fully responsive, includes smooth scrolling, subtle animations, and a light/dark theme toggle.

## Features

- Calm, minimal aesthetic with soft neutral tones
- Component-based sections (Hero, About, Skills, Services, Why Work With Me, Contact)
- Smooth scroll navigation and mobile menu
- Subtle fade-in animations and glassmorphism cards
- Light/dark mode toggle

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Customize

- Replace `[Your Name]` in the UI text with your name.
- Update social links in the Contact section.
- Adjust colors and typography in `tailwind.config.js`.
